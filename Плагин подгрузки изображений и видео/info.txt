Информация о папке animates.

Для плавного скролла страницы можно использовать:

1. Gsap. Но не везде из-за минусов:
  1.1. Если на странице есть аккордеон, при раскрытие сломается страница
  1.2. Не поддерживает position fixed, если элемент с таким position находится непосредственно в контейнере, который используется Scrollsmother для плавного скролла. 
       Пофиксить можно следующим образом: Просто вытащить данный элемент из контейнера, и всё.

2. Библиотеку smooth-scroll.js. Из минусов:
  2.1 Подлагивает при скролле, будто дергается.

Но это не сильно заметно, так как я подобрал оптимальные настройки

3. butter.js. Из минусов:
  3.1. Не поддерживает position fixed, если элемент с таким position находится непосредственно в контейнере, который используется butter.js для плавного скролла.
       Пофиксить можно следующим образом: Просто вытащить данный элемент из контейнера, и всё.


По поводу файла .htaccess:

1) ErrorDocument 404 http://домен сайта/файл с ошибкой.html - перенаправляет на страницу 404, если страница не найдена

2)  RewriteEngine On
RewriteCond %{SCRIPT_FILENAME}/ -d
RewriteCond %{SCRIPT_FILENAME}.html !-f
RewriteRule [^/]$ %{REQUEST_URI}/ [R=301,L]
RewriteCond %{ENV:REDIRECT_STATUS} ^$
RewriteRule ^(.+)\.html$ /$1 [R=301,L]
RewriteCond %{SCRIPT_FILENAME}.html -f
RewriteRule [^/]$ %{REQUEST_URI}.html [QSA,L]
RewriteRule ^index\.html$ / [R=301,L]
RewriteRule ^(.*)/index\.html$ /$1/ [R=301,L]
